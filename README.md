## 💼 Featured Projects

### 🔥 [Chatterly](https://github.com/prio12/Chatterly_Client) - Real-Time Social Media Platform
> **Solo Full-Stack Project** | TypeScript-enabled social platform with real-time messaging, stories, feed, friend system & live notifications

**The Challenge:**  
Most social platforms fail to keep messaging, stories, feed, and notifications perfectly synchronized in real time, leading to delays, inconsistent updates, and broken user experience.

**My Solution:**  
Designed and built a production-ready real-time social media platform using a WebSocket architecture powered by Socket.io. Features include instant messaging with typing indicators and read receipts, Instagram-style stories, a live social feed, friend discovery & requests, and a fully synchronized notification system for posts, comments, likes, messages, and friend interactions.

**Key Technical Achievements:**
- ⚡ Architected a multi-channel WebSocket system managing messaging, notifications, feed updates & presence tracking simultaneously  
- 🟦 Began migrating the frontend to **TypeScript**, converting authentication modules with proper type safety, interfaces, and reusable types  
- 📦 Implemented modular Express.js MVC backend for clean, scalable real-time logic  
- 🎥 Integrated Cloudinary media pipeline with compression achieving ~60% file-size reduction  
- 🐛 Achieved sub-second synchronization with accurate online/offline presence across all user sessions  

**Tech Stack:**  
`React` `TypeScript` `Redux Toolkit` `Node.js` `Express` `Socket.io` `MongoDB` `Cloudinary` `Firebase` `JWT`

🔗 **[Live Demo](https://chatterly-ddcd5.web.app/)** | 💻 **[Frontend](https://github.com/prio12/Chatterly_Client)** • **[Backend](https://github.com/prio12/chatterly_server)**
